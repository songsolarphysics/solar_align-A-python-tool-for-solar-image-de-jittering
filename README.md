# solar_align-A-python-tool-for-solar-image-de-jittering
A Python toolkit for de-jittering solar image sequences. A modern port of J. Chae's IDL routines CH_ALIGNOFFSET and CH_SHIFT_SUB, with extensions for non-cyclic shifts, multi-shape inputs, and offset summaries. Validated on ground-based NVST/Hα and space-based Solar Orbiter EUI/HRI data.
